# c-fundamentos-logica

Estudos e exercícios de lógica de programação em C, com foco no desenvolvimento de base sólida para programação.

## Conteúdo

O projeto está organizado por temas fundamentais:

- **01-condicionais** → estruturas if, else, switch  
- **02-repeticao** → laços for, while, do-while  
- **03-vetores** → manipulação de arrays  
- **04-strings** → operações com texto  
- **05-problemas-classicos** → desafios comuns de lógica  
- **06-validacao-de-entrada** → tratamento de dados do usuário  
- **07-mini-projeto** → aplicação prática integrando os conceitos  

## Objetivo

Consolidar conhecimentos em lógica de programação utilizando a linguagem C, com foco em:
- resolução de problemas
- organização de código
- boas práticas iniciais

## Observação

Este repositório faz parte do meu processo de aprendizado em programação e será atualizado conforme avanço nos estudos.
